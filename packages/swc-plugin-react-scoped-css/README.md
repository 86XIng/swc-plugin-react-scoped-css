# swc-plugin-react-scoped-css

Inspired by [react-scoped-css](https://github.com/gaoxiaoliangz/react-scoped-css)

An react-scoped-css implementation for swc.

## Usage

```js
// .swcrc
{
    "jsc": {
        "experimental": {
            "plugins": [["swc-plugin-react-scoped-css", {}]]
        }
    }
}
```
